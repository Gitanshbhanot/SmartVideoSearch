# Video Analysis Application

This application provides a simple interface for video analysis using YOLO object detection.

## Features

- Upload and analyze video files
- Live webcam analysis
- Object detection using YOLO

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

## Technology Stack

- React
- Material UI
- React Router
- Vite
